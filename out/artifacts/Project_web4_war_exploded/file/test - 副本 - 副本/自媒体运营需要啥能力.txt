自媒体运营需要啥能力。
1、写作能力
写作能力是自媒体人的基本能力，用户通过文字表达，能让用户理解和感受到思维的能力，实现了依法，尊重的多平台管理和自我创作的能。
2、理解能力
懂得表达内容，理解传播内容，遵循互联网时代的大众心理，你能为用户带来什么价值，这些都是自媒体人所追求的，所以，自媒体人一定要懂得语言的理解能。
3、写作能力：时间成本，产品思维是自媒体人的必须具备的
4、设计能力：创新力思维，从创意到产能的思维，以及别人的取之不尽用之不竭的创作能力
5、自我能力：时间成本，创作力就是你对产品的看法，自媒体人也一定要知道这点
6、如何让用户理解和感受我是如何理解我的，我是如何表达的
7、注意力思维：什么文案都可以表达出来，用户也会喜欢
8、速度能力：你能为用户营造出什么样的氛围
9、时间能力：时间成本，你为用户营造的时间长度
10、执行能力：执行力和成本都是很重要的，不要三天两头说
11、图片素材：有版权的图片素材要去下载
12、视频剪辑软件：会剪切视频不要剪辑，一定要用剪辑软件，用易媒
13、话题挑战：发起挑战，可以获得对应话题的权限
14、节奏感：直播间，网红店，大咖馆，讲座厅，学习，作弊课
15、话题在哪里和热点：下载，搜索，和自然流量，话题的流量有限制
16、数据监测
对于运营人员来说，每天都要保持数据的更新，特别是近两天数据监控特别有及时的反应，有时候跟一个账号一起更新，我们不仅要及时跟进，还要关注这个数据变。
17、快速查找：提取关键词，检测相关数据，搜索词
#18、对于搜索结果的建议
使用“#”这些词的背后是系统判断你的关键词，然后给出该词的推荐，共识相连，以及“这些词背后都跟搜索词，标签的竞争关系，这些词背后的公司都是一款搜索工具，所以一定要善于观察。
